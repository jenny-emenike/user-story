User Stories For A Quick Reply App

This is a user story of a prospective chat app which includes details of what the users and site members expect when using the application.

Profile

  • As a user, I want to be able to upload my display picture and give a mini-statement about myself on my profile page.
  • As a user, I need a feature that allows me to either display or hide the time of when last I came online
  • As a site member, whenever I click on a user's display picture, I want his/her details to be displayed, including email and phone numbers.
  • As a user, I want to be able to send my email to other users so that we can connect and communicate.
  • As a site member, I need an icon for making groups to be present in the app. The grpoups are used later on to connect with people of similar interests.

Chat Section

  • As a user, I want all my previous chats to be arranged according to time.
  • As a user, I need a feature that allows me to change the wallpaper of the app according to my preference.
  • As a user, I want emojis and GIFs to also be present in the messgaing chat box.
  • As a site member, I want to be able to upload pictures, videos or any form of multimedia in the chat section,including voice notes.
  • As a user, I need the search box that enables me to search for some previous chats.
  • As a user, I wantg an option for backing up my chats.

Account Details

  • As a user, I want to set a password on the app itself before accessing the app.
  • As a user, I need an option for changing my phone number, email and display pictures.
  • As a user, a delete option needs to be available in case I no longer need the account.
  • As a user, I need a help section or a frequently asked questions(FAQ) section in case I'm having difficulties with the app.

Notifications

  • As a user, I want to set a password on the app itself before accessing the app.
  • As a user, I need an option for changing my phone number, email and display pictures.
  • As a user, a delete option needs to be available in case I no longer need the account.
  • As a user, I need a help section or a frequently asked questions(FAQ) section in case I'm having difficulties with the app.
